# CLEOVERLY Portfolio

Personal portfolio website for **CLEOVERLY**.

## 🌐 Live Website

You can view the live website at: **[cleoverly.online](https://cleoverly.online)**

## 🚀 Features

- **Interactive Waveform**: A canvas-based audio waveform animation that responds to mouse movement speed.
- **Sidebar Navigation**: A rotating wheel-style sidebar menu for desktop navigation.
- **Glassmorphism UI**: UI design utilizing translucent elements and gradients.
- **Responsive Layout**: Adapts to both desktop and mobile screens, including a mobile sliding menu.
- **Skills Showcase**: Grids displaying tech stacks using `react-icons`.

## 💻 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & HTML5 Canvas
- **Language**: TypeScript
